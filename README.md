# Doggo

a sloppy ass client for [janitorAI](https://janitorai.com). goon with ur characters, discover new ones, create your own, and manage personas - all from a mobile app that actually fucking works and isnt just a webview.

built with expo. primary android support with iOS and desktop on the bucket list.

## feats

- **chat:** bare minimum. choose from three chat layouts. toggle narration formatting and timestamps

- **discover:** browse characters. search, filter, sort. view character details n stuff. might find cool stuff

- **create:** bring more slop to the platform!!! drafts should autosave locally btw

- **profile:** settings n stuff, ur characters, ur profile, etc. its better to just try the app out

## setup

get the sloppy [bun](https://bun.sh) and your phone (preferably).

```bash
bun install
bun run android
```
